
# 数字分类工具 📊

一个基于 **Java Swing** 的桌面小工具，帮你快速把文本文件中的数字自动分类。

---

## 功能

读取文本文件（数字间用逗号或换行分隔），自动分为 **6 类**：

| 类别 | 说明 | 示例 |
|------|------|------|
| ✅ 正整数 | 大于 0 的整数 | `42` `7` `100` |
| ✅ 负整数 | 小于 0 的整数 | `-5` `-23` |
| ✅ 正小数 | 大于 0 的小数 | `3.14` `0.5` |
| ✅ 负小数 | 小于 0 的小数 | `-2.71` `-0.618` |
| ✅ 零 | 值为 0 | `0` `-0.0` |
| ✅ 无效数据 | 非数字或格式错误 | `abc` `4a` `1.2.3` |
结果按大小排序，一目了然。
